Key Terms/Ideas

HTML: HypertextMarkup Language
HTTP: Hypertext transfer protocol
Tags: These start and end a string of code, and give it a function to do something
Inline vs. Block: In reference to whether the tag provides an output that is part of a continuous line vs. making an editable box of code.
Inserting Comments: To insert comments into the HTML code, you can use the tag < ! -- (without the spaces)
Head: Element contains information that won't be displayed on the page
Title: Element normally sets the name of the browser tab
Linking a webpage: You can place the following code: a href="www.google.com" TEXT /a
Linking to an image: You can linke to an image by placing an inline tag img src="link to an image.jpg" alt="text" style="something by something"
